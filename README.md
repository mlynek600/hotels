## General info

Hotels ordering app.

## Technologies

Project is created with:

- React
- React Context API
- TypeScript
- Gatsby.js
- styled-components
- mock API / Faker.js
- Axios

## Setup

Clone the project.

Install all packages:

### `npm i`

Start developing, run:

### `gatsby develop`

Open [http://localhost:8000](http://localhost:8000) to view it in the browser.
